# 🤝 Contributing Guide

🌐 Language: **English** | [فارسی](CONTRIBUTING_FA.md)

Thank you for your interest in contributing to this project! ❤️

The goal of this repository is to provide a structured roadmap for learning Python through exercises, projects, and interview questions.

Whether you're fixing a typo, improving documentation, or adding new content, every contribution is appreciated.

---

## 📝 Before You Contribute

Please make sure to:

- Follow the existing project structure.
- Write clean and readable Python code.
- Follow the **PEP 8** style guide.
- Add examples whenever possible.
- Avoid duplicate exercises or projects.

---

## 📚 Adding New Exercises

If you'd like to add a new exercise:

- Place it in the appropriate level:
  - 🟢 Beginner
  - 🟡 Intermediate
  - 🔴 Advanced

- Choose the correct topic.

- Follow the project's exercise format whenever possible.

Each exercise should include:

- Title
- Difficulty
- Problem Statement
- Example Input
- Example Output
- Hint (Optional)
- Solution
- Explanation

---

## 📂 Project Structure

Please keep the existing folder structure.

Example:

```text
01-Beginner/
└── 03-Loops/
    └── 01-For.md
```

---

## 🐞 Issues

Found a bug?
Have an idea?

Please open an **Issue** and describe it clearly.

---

## 🔀 Pull Requests

Before opening a Pull Request:

- Check for duplicate content.
- Make sure your code works correctly.
- Keep your changes focused on a single topic.
- Write a clear description of your changes.

---

## 💡 Ways to Contribute

You can contribute by:

- Adding new exercises
- Improving existing solutions
- Fixing bugs
- Improving documentation
- Correcting typos
- Suggesting better explanations

---

## ❤️ Thank You

Thank you for helping improve this project.

Happy Coding! 🐍
