# 🐍 نقشه راه پایتون

یک مجموعه آموزشی جامع برای یادگیری زبان برنامه‌نویسی پایتون از **سطح مبتدی تا پیشرفته**.

برخلاف بسیاری از مخازن که فقط پاسخ تمرین‌ها را ارائه می‌کنند، هدف این پروژه **یادگیری از طریق حل مسئله** است.
